## Stack

- Node 18.0.0
- Vue 3
- Nuxt 3
- TailwindCSS
- Apollo 5 (GraphQL)
- Pinia 2 (Store)


## How to run

### Install pnpm and dependencies

```bash
npm install -g pnpm
```

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```
